# Portfolio
Professional looking website using Vue.js

![alt text](/src/assets/mdimg.png)



Built using:

* Vue.js
* Emailjs
* JavaScript
* HTML
* CSS

## Setting up

1. Setup directory
```
npm install
```
2. Run the development server
```
npm run serve
```

## Deployment

1. Build files
```
npm run build
```

## Troubleshooting

1. Lint and fixe files
```
npm run lint
```
